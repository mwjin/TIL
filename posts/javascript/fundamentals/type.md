# Type

## Primitive type

Javascript에는 총 *7가지의 primitive type*이 존재한다.

- Number (Integer or Decimal)
- String
- Boolean
- Undefined e.g. `let num; // undefined`
- Null (Empty value) `const a = null;`
- BigInt (ES2015)
- Symbol (ES2015)

## Array type
